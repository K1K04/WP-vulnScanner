Gestión de Claves API

Configura las claves de servicios externos. Los valores se cifran y nunca se muestran en claro.

Empieza por WPScan y Gemini/Claude para tener mejor cobertura de CVEs y funciones IA. El resto de APIs son opcionales y mejoran validación de reputación y alertas.
Para qué sirve cada API

Almacenamiento seguro: Las claves se cifran con Fernet (AES-128-CBC + HMAC-SHA256) derivado de tu SECRET_KEY antes de guardarse en base de datos. Nunca se registran en logs. Las llamadas a esta sección requieren la API_KEY de la aplicación.
